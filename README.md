VoxelNova is a group created by four MD students from Hacettepe University's Computer Animation and Game Technologies program. Repository created for project of a class.
Group Members:

Ugurcan Sonmezoglu - Software

Ege Erdogdu - Software

Canses Demirsoy - Art and Level Design

Levent Postalcioglu - Game Design


The project was done in Unreal Engine 5. 4 unique puzzles were created with different mechanics.
